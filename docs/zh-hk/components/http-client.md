# HTTP Client

The HTTP Client component for Hyperf, from Laravel.

## 安裝

```shell
composer require friendsofhyperf/http-client
```

## 使用

```php
use FriendsOfHyperf\Http\Client\Http;

$response = Http::get('https://example.com');
```
