# About Hyperf Fans

FriendsOfHyperf is a community-maintained collection of packages that extends Hyperf with
production-oriented integrations and Laravel-inspired developer ergonomics.

## Repository Model

The source code lives in the
[friendsofhyperf/components](https://github.com/friendsofhyperf/components) monorepo. Each
directory under `src/` is also released as an independently installable Composer package.
Changes should be submitted to the monorepo rather than to a generated split repository.

## Compatibility

The current branch targets PHP 8.2 or later and Hyperf 3.2. Individual component pages describe
optional integrations and additional dependencies.

## Documentation Languages

Documentation is maintained in English, Simplified Chinese, Hong Kong Traditional Chinese, and
Taiwan Traditional Chinese. Every documentation change must keep the same page paths and heading
structure in all four locales.

## Contributing

Open issues and pull requests in the monorepo. For behavior changes, update the component tests
and all four documentation versions in the same pull request.
